# video-player

Тестовое задание: 

- Базовая верстка страницы
- Запущен видео-плеер на странице с источником типа VOD
- Добавлены дисплеи состояний плеера (текущая позиция воспроизведения, размер буфера)
- Добавлен кастомный плеер (переход через чекбокс на странице)

Стэк: Vue3 (Composition Api), VueUse, Hls

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
